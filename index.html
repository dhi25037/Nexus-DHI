"A, B, C 세 후보의 득표수는 각각 11표, 25표, 14표다. 전체 득표수에서 각 후보의 득표율을 계산해줘. 단계별로 생각하고 설명해줘."
  1. 1단계 (총합 계산): 11 + 25 + 14 = 50표
  2. 2단계 (A후보 계산): (11 / 50) * 100 = 22%
  3. 3단계 (B,C후보 계산): ... 이런 식으로 계속

Beyond the Basics

프롬프트 엔지니어링의 세계는 여기서 끝나지 않습니다.

Zero-Shot CoT

AI에게 "단계별로 생각하자"고 말해 스스로 생각의 연쇄를 유도하는 기술

ReAct

생각하고(Thought), 행동하고(Action), 관찰하며(Observation) 추론하는 프레임워크

Tree of Thoughts

하나의 생각이 아닌, 여러 가능성을 트리 구조로 탐색하여 최적의 답을 찾는 기법
